Ajuste de curvas: Método de Mínimos Quadrados

Introdução

Frequantemente, após uma atividade experimental, obtém-se uma lista de pontos (ti, xi) e deseja-se encontrar a expressão analítica de uma curva x = f(t) que melhor se ajusta ao conjunto de pontos. Essa expressão analítica é denominada equação empírica e o processo pelo qual tal equação é determinada chama-se ajuste de curva.

Por exemplo, considere-se um corpo cuja temperatura Tc no instante de tempo t, imerso num ambiente a temperatura Ta < Tc. A Lei do Resfriamento de Newton afirma que a diferença entre a temperatura do corpo e a temperatura do ambiente diminui exponencialmente em função do tempo. Nesse caso, a diferença x = Tc - Ta decresce com o tempo na forma x = α·eβt (NUSSENZVEIG, 2002). O problema consiste, então, em determinar os valores dos parâmetros α e β que mais se adequem a esta relação exponencial.

Considere-se um conjunto de n pontos (ti, xi). Um dos critérios para o ajuste de curvas é o Método dos Mínimos Quadrados (ZAMBONI; MONEZZI JÚNIOR, 2002). Este método consiste na determinação de qual curva x = f(t) dentre uma família de curvas minimiza a soma:

\[Q = \sum_{i=1}^n (x_i - f(t_i))^2\]
