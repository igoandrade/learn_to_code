Espaços em branco

Por padrão, o HTML ignora espaços em branco. Por exemplo, o seguinte poema Haiku deveria ser mostrado em três linhas:

Life is a journey. Cross the rivers, climb the peaks. But first check for texts.

Uma maneira de dispor o texto na forma desejada é usando a tag pre:

            Life is a journey.
    Cross the rivers, climb the peaks.
        But first check for texts.

Elementos relacionados à código

Para mostrar sintaxe de código, podemos usar as tags:

  • code: indica que o texto é m código de programação;
  • kbd: representa uma entrada para um programa.
  • samp: indica que o texto é uma saída de um programa.
  • var: pode representar tanto uma variável de um programa quanto uma variável matemática.

Character References

Um character reference é um código HTML para mostrar um caractere que de outra forma seria difícil de sr mostrado. Por exemplo, para dizer que uma variável x é menor que zero, podemos usar o código &lt;: x < 0.
